01

Visual Studio Code (VS Code) is a free code editor made by Microsoft. It is the industry standard for writing HTML, CSS, JavaScript, and almost any other language. Installation takes about three minutes and does not require any technical knowledge.

Installation steps

1. Open your browser and go to code.visualstudio.com.

2. Click the large blue Download for Windows button. A .exe file will download to your Downloads folder.

3. Open the downloaded file. If Windows shows a security prompt saying "Do you want to allow this app to make changes?", click Yes — this is expected and safe.

4. Click through the installer screens. When you reach the Select Additional Tasks screen, tick these two options before clicking Next:

  • ☑ Add "Open with Code" action to Windows Explorer context menu — lets you right-click any folder and open it directly in VS Code.
  • ☑ Add to PATH — lets you type code in PowerShell to open VS Code from the terminal.

5. Click Install, wait about 30 seconds, then click Finish. VS Code will open automatically.

💡  If VS Code does not open automatically, find it in your Start menu by searching Visual Studio Code. Pin it to your taskbar so it is always one click away.
02

VS Code opens to a Welcome tab the first time you launch it. A few quick settings will make your day-to-day experience much smoother — none of them require any technical knowledge to configure.

The interface at a glance

VS Code is divided into a few key areas you will use every day:

  • ☰ Activity Bar (left strip of icons) — switches between the file explorer, search, Git panel, and extensions.
  • 📄 Explorer (first icon, looks like two files) — shows all the files and folders in your project.
  • ✎ Editor (main area) — where you read and write code. You can have multiple files open as tabs.
  • ▶ Terminal (bottom panel) — a PowerShell window built into VS Code. This is where you run Git commands and Claude Code.

Recommended settings

Open your project folder. Go to File → Open Folder, navigate to your repository folder, and click Select Folder. VS Code remembers the last folder you opened and will return to it automatically next time.

Enable Auto Save. Go to File → Auto Save and click it to enable it. A tick will appear next to it. This saves your files automatically as you type so you never lose a change by forgetting to press Ctrl + S.

Open the terminal. Go to View → Terminal or press Ctrl + ` (the backtick key, just left of the 1 key). The terminal opens at the bottom of the screen, already pointing at your project folder — you do not need to cd anywhere when you open it this way.

Adjust font size. Hold Ctrl and scroll up or down inside the editor to make text larger or smaller instantly.

Use the Command Palette. Press Ctrl + Shift + P to open a search bar for every feature and setting in VS Code. If you ever cannot find something in the menus, type what you are looking for here.

💡  When you open the terminal inside VS Code, it automatically starts in your project folder — so you do not need to run cd first. This is the easiest way to avoid the "wrong folder" problem described in the Git & GitHub guide.
03

Extensions are add-ons that give VS Code new capabilities. Think of them like apps on your phone — VS Code is the phone, and extensions are the apps you install to do specific things. Each one installs in seconds and can be enabled or disabled at any time. The list below covers the extensions we recommend for this guide.

How to install an extension

Click the Extensions icon in the Activity Bar (the four squares on the left side), search for the extension name, and click Install. Or click the Install in VS Code link on any card below — it will open directly in the Extensions panel.
